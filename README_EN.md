# Object YJS97

An experimental puzzle game built with Apple Metal.

## Introduction

**Object YJS97** is an early Alpha-stage indie game project.  
The game is built using **Apple Metal**, focusing on experimenting with lightweight rendering and puzzle gameplay design.

In the game, the player controls a character called **"Sugar"** and explores different environments.  
By finding keys, activating mechanisms, and solving environmental puzzles, the player gradually approaches the mysterious objective known as **YJS97**.

The project is still in early development and many mechanics may change.

---

## Gameplay

In the game you will:

- Control the character **Sugar**
- Explore puzzle environments
- Find hidden **keys**
- Activate **mechanisms**
- Solve environmental puzzles
- Eventually locate **YJS97**

Some puzzles may involve:

- Path planning
- Mechanism combinations
- Environmental observation
- Logical reasoning

---

## Project Status

Current Version: **Alpha**

The current version mainly focuses on:

- Testing the Metal rendering pipeline
- Basic character movement
- Puzzle mechanics
- Core gameplay experimentation

Future versions may include:

- More levels
