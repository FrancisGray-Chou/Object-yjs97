# Object YJS97

一个基于 Apple Metal 开发的实验性解谜小游戏。

## 项目简介

**Object YJS97** 是一个仍处于 Alpha 阶段的独立小游戏项目。  
游戏采用 **Apple Metal 图形 API** 开发，主要目标是探索轻量级 3D/2D 场景渲染与解谜玩法的结合。

玩家将扮演一个名为 **“糖”** 的角色，在不同的场景中进行探索、解谜，并通过寻找钥匙或触发机关，逐步接近神秘目标 **YJS97**。

目前该项目仍处于早期开发阶段，玩法与机制可能会持续变化。

---

## 游戏玩法

在游戏中你需要：

- 控制角色 **糖**
- 探索关卡场景
- 寻找隐藏的 **钥匙**
- 触发各种 **机关**
- 解开环境谜题
- 最终找到目标 **YJS97**

部分谜题可能涉及：

- 路径规划
- 机关组合
- 场景观察
- 逻辑推理
